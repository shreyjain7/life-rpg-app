# Life RPG App

A gamified personal life tracking application built with Gemini/React.

## About

Life RPG is an interactive application designed to help users track daily habits and personal goals with gamification elements. It transforms routine tasks into an engaging RPG-like experience.

## Features

- **Daily Habit Tracking**: Monitor your daily habits like waking up at 6AM, no snoozing, drinking water, gym workouts, and stretching
- **XP System**: Earn XP points for completing tasks and level up your character
- **Stats Dashboard**: Visualize your progress with a dedicated stats dashboard
- **Grid & Stats Views**: Toggle between grid view (spreadsheet) and stats dashboard
- **Firebase Integration**: Real-time data synchronization and authentication

## Tech Stack

- React with Firebase backend
- React Icons (lucide-react) for UI components
- Chart.js/Bar Chart2 for data visualization
- Material-UI components for styling

## Project Structure

- `app.json` - Main application component and configuration
- Built with React and Firebase
- Uses modern hooks and state management
